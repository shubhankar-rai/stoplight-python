## About The Project

This is a simple python script to simulate the working of a stop-light/traffic-light.


<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shubhankar-rai/stoplight-python.git
   ```

<!-- USAGE EXAMPLES -->
## Usage

This python script is used to simulate the working of a stop-light/traffic-light.

1. Run the python file main.py

2. The resultant will simulate the working of a stop-light infinitely.

3. To quit, simply close the program.
